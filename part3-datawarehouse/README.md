# Part 3: Data Warehouse and Analytics

## Objective
This module focuses on designing a data warehouse for FlexiMart using a
star schema and performing OLAP analytics to analyze historical sales data.

---

## Deliverables

### 1. star_schema_design.md
- Documents the star schema design in text format.
- Describes fact and dimension tables.
- Explains design decisions and sample data flow.

---

### 2. warehouse_schema.sql
- Contains SQL statements to create the data warehouse schema.
- Includes fact and dimension tables as provided.

---

### 3. warehouse_data.sql
- Populates dimension and fact tables with sample data.
- Ensures realistic sales patterns and valid foreign key relationships.

---

### 4. analytics_queries.sql
- Contains OLAP queries for sales drill-down, product performance,
  and customer segmentation analysis.

---

## Evaluation Criteria Alignment

- Star Schema Design (10 Marks) → `star_schema_design.md`
- Schema Implementation (10 Marks) → `warehouse_schema.sql`, `warehouse_data.sql`
- OLAP Analytics Queries (15 Marks) → `analytics_queries.sql`

---

## Conclusion
This part demonstrates dimensional modeling using a star schema and
advanced analytical querying to support business decision-making
in a data warehouse environment.
